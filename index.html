Claude Code 懶人包 #03

讓 Claude 擁有「第二大腦」：自動搜尋、讀取、編輯你的 Obsidian 筆記庫，成為最強教學助理。

Claude Code 核心助理
↔
MCP (mcpvault) 溝通橋樑
↔
Obsidian Vault 筆記庫檔案 (G-Drive)
1

環境準備：Obsidian & Google Drive

Step 0

環境自動檢查

將此懶人包丟給 Claude Code，它會自動檢查：

  • 作業系統版本 (Win/Mac/Linux)
  • Node.js 與 npx 是否可用
  • Google Drive 桌面版路徑
  • Obsidian 安裝狀態
Step 1-4

同步空間與筆記庫

🖐️ 使用者手動操作
  1. 安裝 Google Drive 桌面版 並登入。
  2. 在 Google Drive 磁碟機 (G:\) 建立一個資料夾作為 Vault。
  3. 開啟 Obsidian，點選 Open folder as vault 選擇該資料夾。
2

核心連接：連接 MCP Server

Step 5

安裝與配置 mcpvault

這是最關鍵的步驟，解決桌面版找不到工具的問題：

5-1 全域安裝：
npm install -g @bitbonsai/mcpvault
5-2 寫入設定檔 (三處都要有)：

Claude 會自動將以下設定寫入 ~/.claude/settings.json：

"mcpServers": { "obsidian": { "command": "C:\\Users\\[使用者]\\AppData\\Roaming\\npm\\mcpvault.cmd", "args": ["G:\\我的雲端硬碟\\secondbrain"] } }
Step 6

重啟與驗證

🖐️ 使用者手動操作：關閉並重開 Claude Code 桌面版

輸入指令驗證工具是否出現：

mcp__obsidian__get_vault_stats
3

規則建立：結構化你的大腦

Step 7

建立 CLAUDE.md (班規)

在筆記庫根目錄建立 CLAUDE.md，定義 Claude 的工作習慣：

  • 個人化設定：教什麼科目、什麼年級。
  • 語言偏好：強制使用繁體中文。
  • 寫作規範：自動加上 YAML frontmatter (title, tags, date)。

🚀 之後你可以怎麼用？

你說的話 Claude 會做的事
「搜尋我的筆記有沒有跟 教學遊戲 相關的」 掃描全筆記庫，找出相關內容並摘要給你。
「幫我把今天的對話重點存到筆記」 自動在 vault 建立新 Markdown 檔案。
「幫我整理這篇筆記的重點」 讀取指定檔案內容並分析。

🆘 安裝失敗怎麼辦？

對 Claude 說：「Obsidian 懶人包執行失敗了，幫我檢查哪裡出問題。」

它會自動檢查 mcpvault 路徑與權限，找出斷點。若要重來，可手動清空 settings.json 中的 obsidian 欄位。
